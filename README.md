## 📌 Overview
This project evaluates the performance characteristics of Opportunistic Routing Protocols within the Opportunistic Network Environment (ONE) simulator. Specifically, it simulates a **Tsunami Scenario** to test the resilience and efficiency of Delay-Tolerant Network (DTN) routing protocols, such as Epidemic and Prophet.

## 🚀 Key Features
- **Disaster Scenario Simulation:** Accurately models a Tsunami environment to test network reliability when traditional infrastructure fails.
- **Protocol Comparison:** Directly compares the performance of Epidemic and Prophet routing protocols.
- **Metrics Evaluated:** Analyzes delivery probability, overhead ratio, and average latency to determine the most effective protocol in high-stress environments.

## 🛠️ Tech Stack
- **ONE Simulator:** (Opportunistic Network Environment)
- **Networking:** DTN Routing Protocols (Epidemic, Prophet)
